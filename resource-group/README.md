# Resource Group

## Comandos

1. crear grupo de recursos

```powershell
az group create --name "luiscasalas16-resource-group" --location "eastus2"
```

1. eliminar grupo de recursos

```powershell
az group delete --name "luiscasalas16-resource-group"
```

```powershell
#listar grupo de recursos
az group list
```
